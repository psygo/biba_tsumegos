  answer text one          = {\emph{Correto.} Aplicando um sacrifício, conseguimos reduzir a forma de olho a uma forma morta de 3 espaços. Note que 6 pedras na segunda linha não são o suficiente para viver.},

\problemAnswerDiagram[%
  sgf folder               = vida_ou_morte/olhos_falsos,
  sgf filename             = olhos_falsos.10,
  problem text             = {Preto está no limiar entre a vida e a morte.},
  answer text one          = {\emph{Correto.} Tanto Preto A quanto B funcionariam para crar um espaço de 4 intersecções. Ou seja, 7 pedras na segunda linha vivem, caso Preto jogue primeiro.},
  answer text two          = {\emph{Incorreto.} Jogar em 1 vai sofrer com uma técnica de sacrifício.},
  answer diagram clip vert = 9,
]
